# **App Name**: NumberAce

## Core Features:

- AI Number Guessing Tool: AI powered Number Guessing Tool: Generates a random secret number between 1 and 100, provides hints (higher or lower) after each guess, tracks attempts, and determines rewards based on attempt number.
- Gameplay UI: Interactive Gameplay Interface: A clean, intuitive UI including number input box, hint display, attempts tracker, reward info, and clear 'Play Again' / 'Exit' options.
- Ticketing: Ticketing System: Manages ticket purchases, tracks remaining game credits, and enables 2 solo games per ticket.
- Rewards: Reward System: Calculates and awards points/currency based on the number of attempts taken to guess correctly.
- Game Summary: End-of-Game Summary: Displays the correct number, attempts used, and points/currency earned, with options to play again or exit.
- ReferralBolt: Referral Program (ReferBolt): Pays referral commission, and distributes bonuses and tickets to the user wallet, with tracking of referral cycles. Auto-subscription when the cycle completes.
- Wallet System: Wallet System: Manages user in-app currency, tracks transaction history, and processes deposits/withdrawals, while maintaining fund transaction and transaction history.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of intelligence and trust.  I chose this color because the description of the app highlights intelligence as it involves numerical predictions, logic, strategy, and precision.
- Background color: Light gray (#EEEEEE), desaturated hue similar to the primary, but significantly lighter, providing a clean, neutral backdrop.
- Accent color:  Purple (#9C27B0), an analogous color to deep blue with significantly greater saturation and somewhat greater brightness, to create a focus and emphasize certain elements.
- Body and headline font: 'Inter' sans-serif font providing a clean and modern appearance suitable for all text elements. 
- Use clean, simple icons to represent game elements and actions.
- Maintain a clean, grid-based layout to organize content effectively and ensure readability.
- Incorporate subtle transitions and animations for user engagement and feedback.
